# hermes-youdotcom

Hermes plugin that bundles You.com MCP setup skills for web search, research, finance, and discovery.

## What it includes

- `you-web`: web search, content extraction, and research tools
- `you-research`: focused deep research setup
- `you-finance`: finance and market data setup
- `you-discover`: You.com integration discovery setup
- `you-free`: no-auth web search setup

## Usage

Install the package, enable the `youdotcom` Hermes plugin, then load the bundled skills explicitly from Hermes when needed.
